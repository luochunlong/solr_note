10.1 通过Solrj实现索引添加与更新

通过Solrj实现索引的添加与更新

  • 导入Solrj相关jar包，如下x代表版本号，不同版本的Solr所使用的jar包版本不一致

    • 找到solr解压目录下的dist目录,导入solr-solrj-x.x.x.jar

    • 找到solr解压目录下的dist->solrj-lib目录，导入如下jar文件

  • 自定义solr的字段，在managed-schema文件中添加filed字段

  • 示例代码

    ``` import java.util.UUID; import org.apache.solr.client.solrj.impl.HttpSolrClient; import org.apache.solr.common.SolrInputDocument;

    public class IndexManager {

  public static void main(String[] args) {
      // TODO Auto-generated method stub
      try {
          //创建Solrj客户端对象，通过该对象可以与Solr应用交互
          HttpSolrClient client = new HttpSolrClient.Builder("http://127.0.0.1:8080/solr/core1").withConnectionTimeout(10000).build();

          //创建文档实例，一个文档实例代表一条记录,类似于Lucene中的Document
          SolrInputDocument doc = new SolrInputDocument();
          //通过SolrInputDocument指定列以及值的信息
          //通过UUID生成id值
          String id = UUID.randomUUID().toString();
          //id值不存在则进行添加操作，存在则进行更新操作
          doc.addField("id", id);
          doc.addField("title", "苹果8plus");
          doc.addField("price", 5000f);
          doc.addField("locality", "深圳富士康");
          //添加数据至索引库
          client.add(doc);
          //提交事务
          client.commit();
          //关闭资源
          client.close();
      } catch (Exception e) {
          // TODO: handle exception
          e.printStackTrace();
      }
  }

}

```
